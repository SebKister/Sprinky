# Sprinky

Firmware for a device that controls a sprinkler system, built for Arduino Nano RP2040 Connect via PlatformIO.

## Hardware Setup
*   **Pump Relay**: Pin 2
*   **Inside Valve**: Pin 3
*   **Outside Valve**: Pin 4
*   **Tank Valve**: Pin 5
*   **Inside Switch**: Pin 15
*   **Outside Switch**: Pin 16
